# TirKolija

Управлінський облік транспортного бізнесу: перевізник + експедиція.

- React + Vite, бекенд Supabase (Postgres, Auth, Storage)
- Рейси зі статусами і чек-листом бізнес-процесу
- Витрати з прив'язкою до рейсу або машини, доходи з контролем ПРРО для готівки
- Нарахування ЗП водіям за 4 схемами
- Схема бази: `supabase/schema.sql`

## Запуск
```
npm install
npm run dev
```
